# 🎮 Tetris

A classic Tetris game built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🚀 Demo

Open `index.html` in your browser and start playing instantly. No installation required.

## 🕹️ How to Play

| Key | Action |
|-----|--------|
| `←` `→` | Move piece left / right |
| `↑` | Rotate piece |
| `↓` | Soft drop (faster fall) |
| `Space` | Hard drop (instant fall) |
| `P` | Pause / Resume |

## ✨ Features

- Classic Tetris gameplay with all 7 tetrominoes (I, O, T, S, Z, J, L)
- - Score tracking system
  - - Increasing difficulty as you progress
    - - Next piece preview
      - - Responsive design
        - - No dependencies — pure HTML, CSS & JavaScript
